# Gemini_Chat_Bot
This repository contains a simple question and answer chat bot implemented using the Gemini Pro model. The bot allows users to interact by asking questions and receiving responses generated by the Gemini Pro AI model.
# Setup
# Prerequisites
To run this chat bot, you'll need the following installed on your machine:

* Python (version 3.6 or higher)
* Streamlit
* Google API (Generative AI)
# Installation
1.Clone this repository to your local machine using:
git clone https://github.com/your-username/gemini-pro-chat-bot.git
2.Navigate to the project directory:
cd gemini-pro-chat-bot
3.Install the required dependencies:
pip install -r requirements.txt
# Usage
1.Obtain your API key for the Generative AI and replace the placeholder in the code with your own API key.
2.Run the Streamlit app:
streamlit run app.py
3.Access the chat bot in your web browser at http://localhost:8501.
4.Start the conversation by sending a message to begin chatting with the Gemini Pro bot.
# Important Note:
Make sure to use your own API key in place of the placeholder provided in the code to access the Generative AI. Do not share your API key publicly.
# How it Works:
1.The chat bot utilizes the Gemini Pro AI model to generate responses based on user input.
2.Upon launching the Streamlit app and starting a conversation, messages are exchanged between the user and the bot.
3.The chat history is recorded using Streamlit session state, displaying the conversation log for reference.
